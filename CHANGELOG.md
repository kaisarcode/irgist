# Changelog

## v2.0.0

* Redefined IRGist as a general semantic IR compressor for arbitrary text.
* Replaced the v1 prompt encoder with the semantic memory compression prompt.
* Expanded scope from prompt execution gists to LLM context reuse.
* Added positioning for documentation, agent instructions, stories, specs, chats, and project memory.
* Documented the one-line semantic memory record format.

## v1.0.0

* Initial public version of IRGist.
* Provided a prompt encoder for compact execution gists.
* Focused on natural-language instructions for domain-specific LLM tools.
